🚀 一鍵同步到 GitHub Pages
🔑 如何取得 GitHub Token？（點此展開）
Step 1 登入 GitHub → 右上角頭像 → Settings
Step 2 左側最底部 → Developer settings
Step 3 Personal access tokens → Fine-grained tokens
Step 4 Generate new token，填寫：
  • Token name：plant-map-sync（隨意）
  • Expiration：No expiration 或 1 year
  • Repository access：Only select repositories → 選你的 campus-plant-map
  • Permissions → Contents → Read and write
Step 5 Generate → 複製 token（只顯示一次！）貼到下方
🌐 分享到網路（免費）
把植物地圖分享給學生、家長或任何人，只需三個步驟：
Step 1 按下方「匯出資料」，下載 campus-plant-data.json
Step 2 前往 github.com 建立免費帳號，新增一個 Public 儲存庫
Step 3 上傳以下三個檔案到儲存庫根目錄：
  • index.html（本程式主檔）
  • campus-plant-data.json（你的植物資料）
  • manifest.json · icon.svg（PWA 圖示）
Step 4 到 Settings → Pages → Branch: main → Save
完成！網址：https://你的名字.github.io/儲存庫名稱/

🔄 更新資料：每次修改植物後，重新匯出 JSON 並上傳覆蓋即可。
